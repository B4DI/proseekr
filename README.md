# ProSeekr v1.0.1

A bi-lingugal job portal for blue-collars.

### Features
- Bi-lingual support (Hindi and English)
- Text to Speech (TTS) support in both the languages
- Push and In-app notifications 
- Simple User Interface
- Minimal User flow
